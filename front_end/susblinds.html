How to use interactive diagram:

  • Press play to open interactive diagram

  • Use slider to adjust the time of day, from 4am - 8pm.

  • Select the month and the direction the window is facing using the select boxes.

  • Input outside tempeture for suggested blind position.

  • Press the Calculate button to find how much heat gain the window is providing
